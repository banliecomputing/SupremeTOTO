Ingin Database Permanen Milik Sendiri?

Saat ini Anda menggunakan Cloud Sementara. Untuk membuat database Anda tidak pernah hilang, ikuti langkah ini:

  • Buka Firebase Console, klik Create Project (bebas nama).
  • Pilih menu Firestore Database > Create > Start in Test Mode.
  • Pilih menu Authentication > Get Started > Anonymous (Aktifkan).
  • Ke Project Overview (Home), klik logo Web </> untuk mendaftarkan web.
  • Copy kode di dalam firebaseConfig = {...}.
  • Tutup menu aplikasi ini, lalu Paste/Tempel kode tersebut ke baris ke-430 di Editor Kode (Layar Kanan).
